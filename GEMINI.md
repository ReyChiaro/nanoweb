# Project Context: Personal Website

## Overview
This repository hosts the source code for a personal portfolio website. The primary goal is to build a fast, accessible, and visually appealing site to showcase professional projects, share blog posts, and provide contact information.

## AI Agent Guidelines (Prompts & Context)
When working on this project, please adhere to the following context and constraints:

### Tech Stack & Architecture
*   **Framework:** Vue 3
*   **Styling:** CSS Modules
*   **Language:** TypeScript
*   **Package Manager:** npm

### Coding Standards & Style
*   **Component Structure:** Functional components with hooks. Keep components small and reusable.
*   **Styling:** Mobile-first responsive design. Dark/Light mode support preferred.
*   **Accessibility:** Semantic HTML (headings, landmarks, alt text) is mandatory.
*   **Naming:** PascalCase for components, camelCase for functions/variables.

### Workflow & Conventions
*   **Commits:** Use Conventional Commits (e.g., `feat: add about page`, `fix: mobile menu layout`).
*   **Testing:** Write unit tests for utility functions and integration tests for critical flows.

## Operational Commands
*   `npm run dev`: Starts the local development server.
*   `npm run build`: Builds the project for production.
*   `npm run lint`: Runs code linting.
